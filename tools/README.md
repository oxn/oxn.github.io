# cytools-html 更新日志 v0705（2026-07-05）
## 版本说明
旧版：v0704（2026-07-04）
新版：v0705（2026-07-05）
本次为底层架构+多浏览器兼容+性能大更新，统一全平台字体渲染、标准化CSS公共类、JS逻辑缓存优化，修复多款浏览器视觉差异与页面卡顿问题。

## 标准数据模版
```js
{
  "id": "工具唯一ID",
  "title": "工具名称",
  "icon": "Emoji图标",
  "desc": "工具功能简短描述",
  "cat": "分类标识",
  "date": "更新日期 格式YYYY-MM-DD",
  "featured": true/false, // 外链工具标记
  "top": true/false, // 置顶工具
  "url": "页面地址",
  "target": "_blank", // 仅外链填写，本地工具省略
  "tags": ["标签1","标签2","标签3"]
}
```

## 一、核心更新：全平台统一字体渲染（解决Chrome/360显示不一致）
1. 重构全局字体栈，**微软雅黑前置**，Windows优先原生微软雅黑；Mac/iOS自动使用系统苹方；安卓适配系统无衬线字体。
2. 更换渲染模式为 `subpixel-antialiased`，移除灰度抗锯齿，修复360浏览器小字发灰、模糊、对比度不足问题。
3. 全局 `*` 选择器增加 `font-family: inherit`，按钮、输入框、卡片、标签全部统一字体，彻底消除局部宋体错乱。
4. 删除翻车属性 `font-synthesis: none`，浏览器正常渲染600/700粗体标题、导航按钮，标题不再单薄发虚。

## 二、CSS架构重构，公共样式标准化，大幅精简冗余代码
### 1. 新增全局玻璃 `.glass` 公共类
- 头部、搜索框、分类按钮、卡片、收藏按钮、页脚全部复用，删除数十处重复玻璃模糊、边框、阴影代码。
- 亮色/暗色两套玻璃样式统一维护，一处修改全局同步生效。
### 2. 统一悬浮交互 `.hover-float` 通用动画类
- 全部可点击组件（导航、分类、卡片、排序、收藏星）共用一套上浮/按压动画，交互手感完全统一。
- 内置禁止双击选文字 `user-select: none`，优化操作体验。
### 3. 新增辅助工具类 `.no-select`
快捷键提示、静态装饰标签专用，防止拖拽选中文字干扰操作。
### 4. CSS分层规范化
代码按固定分层排序：CSS变量 → 暗色主题覆盖 → 公共复用类 → 全局重置 → 页面模块，结构清晰便于长期维护迭代。

## 三、工具卡片视觉细节精细化优化
1. **卡片描述两行省略稳定优化**
    - 固定行高 `1.4`，统一卡片最小/最大高度，解决不同长度描述导致卡片高低不齐。
    - 补全全套多行省略兼容属性，文字截断稳定无溢出、无错乱。
2. **底部标签单行省略**
    - 标签容器禁止自动换行，标签过多自动横向省略，不再撑高卡片破坏网格布局。
    - 单个标签 `white-space: nowrap`，标签文字不会折行变形。
3. **收藏星星交互升级**
    - 卡片默认隐藏收藏按钮，hover后显示，界面更整洁清爽。
    - 收藏按钮复用统一玻璃悬浮样式，点击阻止冒泡，不会误触卡片跳转。
4. 统一卡片图标边框透明度，明暗主题自动适配切换。

## 四、JS底层性能大幅优化，页面加载、筛选更流畅
1. **精简工具列表数据结构**
    - 新增全局分类映射 `CAT_MAP`，工具项移除冗余 `catName`、`catIcon` 字段，页面体积更小、加载更快。
    - 自动合并重复分类（TXT/TEXT统一归类），内置未知分类兜底规则，无需重复配置。
2. **筛选结果缓存机制 `filterCache`**
    搜索关键词+分类+排序组合结果缓存，切换分类、输入搜索不再循环遍历全部工具，页面响应速度显著提升。
3. **分类标签宽度永久缓存**
    仅首次加载测量标签宽度，窗口缩放仅清空缓存、不重复DOM计算，减少页面重绘卡顿。
4. **分片渲染，不阻塞主线程**
    使用 `requestIdleCallback` 分批次渲染卡片，上千条工具列表也不会页面卡死。
5. 交互细节优化
    - `Ctrl + K` 聚焦搜索框时自动全选文字，一键覆盖输入。
    - 区分空白提示文案：无匹配工具 / 暂无收藏，提示更精准。
    - 切换分类、收藏操作自动清空缓存，数据实时同步刷新。
6. 防抖逻辑优化，减少无效渲染与重绘开销。

## 五、页面DOM结构标准化改造
1. Header、搜索框、分类按钮、排序按钮、页脚统一添加 `.glass` 公共类，删除重复内联玻璃样式。
2. 所有可点击交互元素统一挂载 `.hover-float`，悬浮动画统一管控。
3. 快捷键提示标签添加 `.no-select`，禁止文字拖拽选中。
4. 页脚文案精简，更新日期同步更新为新版本时间。

## 六、明暗主题配套同步适配
1. 玻璃面板、悬浮边框、阴影亮色/暗色两套主题同步修正。
2. 收藏按钮激活态边框色彩适配暗色模式。
3. 保留随机渐变背景逻辑，切换主题自动重新生成背景。

## 七、响应式布局加固

3. 三栏/两栏/单栏网格断点稳定，卡片间距、最小宽度全部使用CSS变量统一管控。

## 八、本次修复旧版已知问题
1. 修复360极速浏览器字体粗细、灰度显示和Chrome不一致问题。

7. 修复点击收藏星星同时触发卡片跳转的冒泡冲突。
8. 修复空白提示文案模糊，无法区分无搜索结果、无收藏两种场景。

## 九、长期维护性提升
1. 大量重复玻璃、hover、阴影、边框样式抽离公共类，CSS文件体积精简。
2. 全部圆角、间距、阴影、过渡动画使用CSS变量统一管控，后期改版仅修改变量区。
3. JS逻辑分层清晰：主题、分类、筛选、渲染、收藏、工具数据分离，新增工具/分类无需改动底层逻辑。
4. 去除多处重复DOM拼接代码，统一渲染模板，减少冗余代码。
